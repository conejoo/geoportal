### How do I get set up? ###


Install Proyect:


```
#!bash

clone git@github.com:conejoo/geoportal.git
cd geoportal
bower update
```


### How to run project in development ###


```
#!bash

cd <project_folder>/public
python -m SimpleHTTPServer

```
